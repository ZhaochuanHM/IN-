# xiao_injiu
php + wx小程序。
# 一 从搭建到开发：
  1、在小程序开发注册：在微信公众平台注册小程序，完成注册后可以同步进行信息完善和开发。<br>
  2、小程序信息完善：填写小程序基本信息，包括名称、头像、介绍及服务范围等。<br>
  3、开发小程序：完成小程序开发者绑定、开发信息配置后，开发者可下载开发者工具、参考开发文档进行小程序的开发和调试。<br>
  4、提交审核和发布：完成小程序开发后，提交代码至微信团队审核，审核通过后即可发布（公测期间不能发布）。<br>
